# Feature: Mobile Optimization for Landing Page

## Overview
This feature focuses on making the landing page (`index.html`) fully responsive, adhering to the "Mobile-First" strategy defined in the Cryptid Console design system. It introduces a mobile-specific navigation menu and responsive typography.

## Scope
- Update `src/public/index.html` with responsive Tailwind classes.
- Implement a mobile-only "System Tray" navigation menu.
- Ensure the "Terminal Frame" hero viewer is fluid on small screens.
- Enhance accessibility with proper touch targets.
