# Implement "Cryptid Console" Design System

## Overview
This feature implements the "Cryptid Console" design system on the landing page (`src/public/index.html`). The goal is to replace the existing simple image display with a themed interface featuring a navigation bar and a "terminal-style" image viewer.

## Scope
- Update `src/public/index.html` to use Tailwind CSS via CDN (as per styleguide).
- Implement the "Cryptid Console" color palette, typography, and component styles.
- Preserve the existing "secret" 3-click navigation to `/app`.
- Add a visible Login button in the navigation bar.
